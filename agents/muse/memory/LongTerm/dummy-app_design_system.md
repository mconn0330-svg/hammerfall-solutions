# LongTerm Memory — dummy-app Design System
**Project:** dummy-app
**Date:** 2026-03-27

## Aesthetic Directives
- **Theme:** Terminal-Native.
- **Typography:** `monospace` exclusively. High contrast text.
- **Vibe:** Raw, mechanical speed. It must look and feel like executing a command line tool, translated into a modern Next.js/Tailwind web app.

## UX Constraints
- **Keyboard-First:** Mouse-friendly but entirely keyboard navigable. Actions are mapped to keystrokes (`x` complete, `e` edit, `d` delete, `j`/`k` movement).
- **Core Input:** The primary interface is a Global Command Prompt with Natural Language Parsing (e.g., `> add Deploy database migrations #devops !high`).
- **Data Model UI Mapping:** Flat lists only. No complex nested hierarchy. No avatars, no team assignment badges, no comment threads. It is explicitly hostile to collaborative features.
